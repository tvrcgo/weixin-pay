# weixin-pay

微信支付 API Nodejs SDK

[![npm version](https://badge.fury.io/js/weixin-pay.svg)](http://badge.fury.io/js/weixin-pay)

## Features

- 基于微信官方 API v3
- 基础支付支持 JSAPI/H5/App/Native/小程序


## Install

```bash
$ npm install weixin-pay --save
```
